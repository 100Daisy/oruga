# Examples

## Tailwind CSS

<p>
Using override mode is easy to integrate a CSS framework with Oruga, just define your custom classes and relate them by config constructors.
<br>
Hovewer you should evaluate to use <code>orga-lite.css</code> in order to avoid to define the css structure of Oruga components (display, position, z-index and other base attributes).
<br>
Source code <a href="https://github.com/oruga-ui/oruga/tree/master/packages/oruga">here</a>
